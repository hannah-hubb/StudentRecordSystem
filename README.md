Student Record Management System

Overview

A simple Java project for managing student records. The program allows users to add, view, search, update, delete student information, and store data using Java File I/O and object serialization.

Features

* Add student records
* View all students
* Search for a student by ID
* Update student information
* Delete student records
* Save and load data using Java File I/O

Reporting

The system generates basic statistics from the stored student data:

* Total number of students
* Highest GPA
* Lowest GPA
* Average GPA

Backup

* The system creates a backup file (backup.dat)
* Uses BufferedInputStream and BufferedOutputStream
* Ensures student data can be restored if the main file is corrupted or lost

Exception Handling

All file operations are protected using try-catch blocks.

The system handles:

* IOException
* ClassNotFoundException
* FileNotFoundException

This ensures the program does not crash during file operations and provides safe error handling.

Contributors

This project appears under two contributors due to development being done on a shared computer environment.

However, all design, implementation, and development work was completed independently by the primary author. The additional contributor name is automatically generated by the system due to the use of another user account on the device.

How to Run

1. Compile all Java files
2. Run Main.java
3. Use the menu to manage student records

Purpose

This project was created to practice Object-Oriented Programming, Java File I/O, exception handling, and data persistence concepts.
